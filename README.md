# ⚽ FootballVerse

A modern, premium social media platform for football enthusiasts inspired by Pinterest. Share, discover, and celebrate football culture with a sleek black, white, and gold design.

## 🎯 Features

### Core Features
- **Home Feed**: Discover football content from users you follow
- **User Authentication**: Sign up and login functionality
- **User Profiles**: Customizable profiles with pictures and bios
- **Media Upload**: Share football images and content
- **Interactions**: Like, comment, save, and share posts
- **Social Connections**: Follow other users and build communities
- **Search**: Find players, clubs, leagues, and users
- **Trending Page**: Discover trending football content
- **Categories**: Organized content in 7 football categories
  - Wallpapers
  - Kits
  - Boots
  - Tactics
  - Stadiums
  - Edits
  - Memes

### Design Features
- **Premium Design**: Black, white, and gold color scheme
- **Mobile-First**: Fully responsive design for all devices
- **Smooth Animations**: Professional transitions and interactions
- **Pinterest-Inspired Layout**: Masonry grid layout for optimal content display
- **Modern UI**: Contemporary interface similar to Pinterest but unique to football

## 🛠️ Tech Stack

### Frontend (Complete)

## 🚀 Getting Started

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ndumi12394/FootballVerse.git
   cd FootballVerse
   ```

2. **Open in browser**
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     python -m http.server 8000
     # or
     npx serve
     # or
     php -S localhost:8000
     ```

3. **View the application**
   - Navigate to `http://localhost:8000`
